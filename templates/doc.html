Tecnologias e ferramentas utilizadas

Flask

"Microframework para Python baseado em Werkzeug, Jinja 2 e boas intenções." Foi a plataforma escolhida para desenvolvimento do site.

PythonAnywhere

"Execute e codifique Python na nuvem!" Foi o modo escolhido para dar o deploy do site em produção.

GitHub

Plataforma de hospedagem de código-fonte com controle de versão que utiliza o Git. Local escolhido para fazer o versionamento do site.

Visual Studio Code

Editor de código-fonte, tão poderoso quanto uma IDE, open source desenvolvido pela Microsoft. Através dele que o código foi escrito.
